TuPrimeraPagina_Acosta

 Este proyecto implementa una web tipo Blog utilizando el patrón MVT (Modelo-Vista-Template).

Funcionalidades

El proyecto incluye las siguientes características:

Herencia de Plantillas HTML: Uso de un archivo base (padre.html) para mantener un diseño consistente.

Gestión de 3 Modelos: Autor, Categoria y Post.

Formularios: Inserción de datos para cada modelo.

Búsqueda: Funcionalidad para buscar autores por nombre en la base de datos.

Orden de Pruebas Sugerido

Para verificar el correcto funcionamiento del sistema, se recomienda seguir este flujo:

Crear Autor: Ir a la sección "Nuevo Autor" y registrar uno (ej: Juan Perez).

Crear Categoría: Ir a "Nueva Categoría" e ingresar un nombre (ej: Tecnología).

Crear Post: Ir a "Nuevo Post" y publicar un artículo.

Buscar Autor: Ir a la sección "Buscar" e ingresar el nombre del autor creado previamente (ej: Juan) para verificar que el sistema lo encuentra.
